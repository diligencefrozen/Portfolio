## 1. CSS 기본 개념

### CSS(Cascading Style Sheet)

HTML DOM 객체에 디자인을 부여하는 언어이다. 즉, 웹페이지에서 사용자가 보는 시각적 표현(UI, 사용자 인터페이스)을 꾸미는 역할을 한다.

---

## 2. CSS 작성 위치

1. `.html` 파일의 `<head></head>` 안
2. DOM 객체의 속성 형태로 직접 작성
3. `.css` 파일을 따로 만들어 작성 → **가장 많이 사용하는 방식**

---

## 3. CSS 기본 문법

```css
선택자 {
    속성명: 값;
    속성명: 값;
}
```

- **선택자(selector)**: 어느 부분을 꾸밀지 지정
- **속성명**: 적용할 디자인 종류
- **값**: 실제 적용 값

---

## 4. 선택자(selector)

### 의미

CSS로 UI를 꾸밀 때, **어느 요소를 디자인할지 지정하는 것**이다.

### 4-1. 기본형 선택자

### ① 태그 선택자(type 선택자)

```css
div {
    color: #0000FF;
}
```

→ 모든 `div` 태그 선택

### ② id 선택자

```css
#holiday {
    background-color: yellow;
    width: 300px;
    height: 300px;
}
```

→ `id="holiday"`인 요소 하나 선택

- **id는 중복 사용 X**

### ③ class 선택자

```css
.c {
    font-weight: 900;
}
```

→ `class="c"`인 요소들 선택

- **class는 중복 사용 O**

### 4-2. 조합형 선택자

### ① 그룹 선택자

```css
h1, h2 {
    text-decoration: overline;
}
```

→ `h1`, `h2`를 동시에 선택

### ② 하위 선택자

```css
div h3 {
    text-shadow: orange 3px 3px 5px;
}
```

→ `div` 내부의 `h3` 선택

- 그림자 형식: **색상 x축 y축 번짐효과**

### ③ 종속 선택자

```css
h3.c {
    border: #99FFFF groove 7px;
}
```

→ `h3`이면서 `class="c"`인 요소 선택

- `border`: **색상 선종류 두께**
- 선 종류 예시: `solid`, `dashed`, `dotted`, `double`, `inset`, `outset`, `groove`

### ④ 자식 선택자

```css
div > h4 {
    color: red;
}
```

→ `div` 바로 아래 자식 `h4` 선택

### ⑤ 인접 선택자

```css
div + h5 {
    color: #CC00A1;
}
```

→ `div` 바로 뒤에 나오는 첫 번째 `h5` 선택

### 4-3. 상태형 선택자(가상 클래스 선택자)

```css
#btn:active {
    border: #FF6666 inset 5px;
}

#btn:hover {
    font-family: "궁서";
}
```

- `:active` → 클릭 중일 때
- `:hover` → 마우스를 올렸을 때
- `:focus` → 입력창에 커서가 들어간 상태

## 5. cssTest.html 핵심 정리

### 사용된 선택자들

- `div` → 태그 선택자
- `#holiday` → id 선택자
- `.c` → class 선택자
- `h1, h2` → 그룹 선택자
- `div h3` → 하위 선택자
- `h3.c` → 종속 선택자
- `div > h4` → 자식 선택자
- `div + h5` → 인접 선택자
- `#btn:hover`, `#btn:active` → 상태형 선택자

### 블록 요소와 인라인 요소

- `div` → **블록 요소(block element)**, 한 줄 전체 차지
- `span` → **인라인 요소(inline element)**, 필요한 영역만 차지

## 6. index.css 핵심 개념

### `@charset "UTF-8";`

CSS 파일 문자 인코딩 지정

### `@font-face`

웹폰트 등록

- 폰트 파일을 CSS에 연결해 사용
- 용량이 커질 수 있어 느려질 수 있음

### `::-webkit-scrollbar`

스크롤바 숨김

### `margin` / `padding`

- `margin` = 바깥 여백
- `padding` = 안쪽 여백

### `border-spacing`

테이블 셀 사이 간격 조절

### `transition`

애니메이션 효과

```css
transition: 속성명 시간 ease-in/ease-out;
```

### `text-shadow` / `box-shadow`

- `text-shadow` = 글자 그림자
- `box-shadow` = 요소 자체 그림자

### 가운데 정렬

```css
margin-left: auto;
margin-right: auto;
```

## 7. diary.css 핵심 정리

```css
#diaryTbl {
    width: 990px;
    background-color: #FFCC00;
    font-size: 20pt;
    border-spacing: 0;
}

#diaryTbl .dataTr {
    background-color: white;
    color: #ffaaaa41;
    height: 40px;
    cursor: pointer;
}

#diaryTbl th {
    height: 60px;
    font-size: 20pt;
}

#diaryTbl td {
    text-align: center;
    width: 33%;
    border-bottom: black solid 2px;
}

#diaryTbl .dataTr:hover {
    background-color: #990033;
}
```

### 핵심

- `th` = 제목 셀
- `td` = 일반 셀
- `cursor: pointer` = 클릭 가능한 느낌
- `border-bottom` = 아래 테두리
- `:hover` = 마우스를 올렸을 때 색 변화

## 11. 최종정리

### CSS 한 줄 정의

**CSS는 HTML 요소에 디자인을 적용하는 언어이다.**

### CSS 작성 위치 3가지

1. HTML의 `<head>` 안
2. 태그 속성에 직접 작성
3. 별도 `.css` 파일 작성 → 가장 많이 사용

### 문법

```css
선택자 {
    속성명: 값;
}
```

### 기본 선택자

- `태그명` → type 선택자
- `#id명` → id 선택자
- `.class명` → class 선택자

### 조합 선택자

- `선택자, 선택자` → 그룹 선택자
- `선택자 선택자` → 하위 선택자
- `선택자.class명` → 종속 선택자
- `선택자 > 선택자` → 자식 선택자
- `선택자 + 선택자` → 인접 선택자

### 상태 선택자

- `:hover`
- `:active`
- `:focus`

### 자주 나오는 속성

- `color` = 글자색
- `background-color` = 배경색
- `width`, `height` = 크기
- `font-size` = 글자 크기
- `font-weight` = 굵기
- `text-decoration` = 윗줄, 밑줄 등
- `border` = 테두리
- `text-shadow` = 글자 그림자
- `box-shadow` = 요소 그림자
- `margin` = 바깥 여백
- `padding` = 안쪽 여백
- `text-align` = 정렬
- `overflow` = 넘침 처리
- `white-space` = 줄바꿈 처리
- `text-overflow` = 넘치는 글자 처리

### 블록 / 인라인

- `div` = 블록 요소, 한 줄 전체 사용
- `span` = 인라인 요소, 필요한 만큼만 사용

### 테이블

- `th` = 제목 셀
- `td` = 일반 셀
- `rowspan="2"` = 세로 2칸 합치기

### textarea

- `readonly` = 수정 불가
- `resize: none` = 크기 조절 금지
- `outline: none` = 포커스 외곽선 제거

### 반드시 구분

- `text-shadow` ↔ `box-shadow`
- `margin` ↔ `padding`
- `id` ↔ `class`
- `div` ↔ `span`

### 초압축

**CSS = HTML 꾸미기**

**문법:** `선택자 { 속성명: 값; }`

**선택자:**

태그 / id / class / 그룹 / 하위 / 종속 / 자식 / 인접 / 상태형

**핵심 개념:**

- `div` = 한 줄 전체
- `span` = 필요한 만큼
- `hover` = 마우스 올릴 때
- `active` = 클릭 중
- `focus` = 입력 중

**실수 포인트:**

- `border-radius`
- `ellipsis`
- 파일명/클래스명 정확히 맞추기
