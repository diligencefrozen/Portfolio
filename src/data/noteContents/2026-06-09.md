```python
def hJ():
    name = input('이름을 입력:')
    print("안녕하세요. ",name,"님! 환영합니다.")

    comHand = random.randint(1, 100)
    attempt = 0
    max_attempts = 10
    # 유저에게 정답을 입력하게 했을 때
    while attempt < max_attempts:
        attempt += 1
        try:
            
            userAnswer = input('값을 기입하세요:')

                print('1~100 사이 값 기입:')
                attempt -= 1
                continue
            
            if userHand < comHand:
                print("UP!")
            elif userHand > comHand:
                print("DOWN!")
            else:
                print(f'정답! 총',attempt,'번 만에 맞춤.')

        except ValueError:
```

## 람다 (lambda) 함수

장점 : 코드 간결, 가독성 향상

- 일반 함수(def)와 동일하게 함수 객체로 동작
- (특별한 메모리 절약은 없음)
- 표현법 :
- lambda 매개변수(=파라미터):

단점 : 한 줄 짜리 표현식만 가능, 여러 문장은 불가능

ex)

여러 문장을 넣으려 하면 문법 오류 발생

#### 잘못된 사용법 예시 ㄱ

```python
wrong_lambda = lambda x:
    y = x +1
    return y
```

#### 변수를 할당하는 일반적인 함수

```python
def hab(num):
    x = num + 100
    return x
print(hab(100))
```

ㄴ 이 코드 수를 줄여주는 방법이 있는데, 그게 람다함수임.

