# 2. throws InterruptedException

```java
public static void main(String[] args) throws InterruptedException
```

설명

→ 서버 자원을 낭비하지 않기 위한 조치

`throws InterruptedException` 은 **sleep 중 발생할 수 있는 예외를 호출한 쪽으로 넘긴다**는 의미이다.

# 4. 리뷰 창 출력 (printf)

내용

- 집까지 걸린 시간
- 저녁 식사 메뉴
- 수업 후 활동
- 취침 시간
- 소감
- 수업 성취도

```java
System.out.printf("%s\n","############ 리 뷰 창 ############");

System.out.printf("%s\0", "집으로 돌아가는 데 소요된 시간은 총 얼마가 걸렸나요? :");

System.out.printf("%d분\0", 60);

System.out.printf("%1f시간\n", 1.3);
```

설명

→ 소수점은 `\f` 사용

소수점 서식 지정은

```
%f
```

이다.

예

```
%.1f
%.2f
```

---

```java
System.out.printf("%s\n", "저녁 식사 메뉴는 무엇이었습니까? : 한식");

System.out.printf("%s\n", "수업 이후 무엇을 했나요? : 회화 연습");

System.out.printf("%s\0", "취침 시간 :");

System.out.printf("%d시%d분\n", 10, 01);

System.out.printf("%s\n", "소감: 좋은 연습시간이 되어 굉장히 영광스럽게 생각합니다.");

System.out.printf("%s\0", "수업 성취도 :");

System.out.printf("%f%%\n", 98.99);

System.out.printf("%s\n", "########### 리 뷰 창 #############");

Thread.sleep(10000);
```

→ 10초 딜레이

# 14. 제품 설명서 실습

제품

```
섬유향수
가격 1000원
용량 80ml
체리블라썸 향
항균 99.9%
무색
다이소 제작
재활용 여부
```

변수

```java
String prod1 = "섬유향수";
int price = 1000;
int detail01 = 80;
String smell = "체리블라썸";
double effect = 99.9;
String colour ="무";
String maker = "다이소";
boolean earthkeeper2 = true;
```

출력

```java
System.out.printf("가격(원) : %d원\n", price);
System.out.printf("용량 : %dml\n", detail01);
System.out.printf("종류 : %s 향\n", smell);
System.out.printf("항균 : %.1f%%\n", effect);
System.out.printf("제품의 색깔 : %s색\n", colour);
System.out.printf("이 제품의 생산지 : %s\n", maker);
System.out.printf("재활용 필수인 제품인가? : %b\n", earthkeeper2);
```
