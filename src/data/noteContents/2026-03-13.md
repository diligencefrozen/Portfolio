## 1. `for문` 기본 개념

### 기본 구조

```java
for (변수초기화; 조건식; 값의증감) {
    내용
}
```

### 핵심 개념

- 반복 **횟수가 명확할 때** 주로 사용
- 시작값, 조건, 증가/감소를 한 줄에 표현 가능
- 특정 횟수만큼 반복하기에 적합

## 2. Up & Down 게임 예제 (`for문` 활용)

### 개념

- 컴퓨터가 1~100 사이의 정수를 하나 정함
- 내가 숫자를 입력
- 정답보다 크면 `Down!`
- 정답보다 작으면 `Up!`
- 맞히면 종료
- 몇 턴 만에 맞혔는지 출력

### 코드

```java
import java.util.Random;
import java.util.Scanner;

public class RMain1 {
// Up & Down 게임 (main에)
// 1 ~ 100 사이의 정수
// 컴퓨터 vs 나 => 같은 숫자를 내면 종료
// 정답이 나보다 낮으면 'Down!'
// 정답이 나보다 높으면 'Up!'
// 몇 턴만에 정답을 찾았는지 출력!
// 반복문(for), 조건문(if)

	public static void main(String[] args) {

		Random r = new Random();
		Scanner s = new Scanner(System.in);
		int num_pc = r.nextInt(100)+1;

		int num_myself = 0;
	    //int num_pc = r.nextInt(99)+1;

		// 몇 번 반복할지 모르기 때문에 조건을 지정할 수 없음.
		// true값을 넣어서 무한반복 시킬수도 있다.
		for (int turn = 1; true; turn++) {
			System.out.println("정수 입력:");
			num_myself = s.nextInt();
			if (num_pc == num_myself) {
				System.out.println("정답!");
				System.out.printf("%d번만에 정답을 맞춤.", turn);
				break; // 반복문을 깨버림
			} else if(num_pc < num_myself) {
				System.out.println("Down!");
			} else  {
				System.out.println("Up!");
			}
			System.out.println();

		}
	}
}
```

### 정리 포인트

- 반복 횟수를 모를 때도 `for (;;)` 또는 `for (...; true; ...)` 형태로 무한 반복 가능
- `break;`를 만나면 반복문 즉시 종료
- `turn` 변수로 시도 횟수 관리 가능

## 3. `for문` 예제 1: 1~100 사이의 6의 배수 합

# 6. `while문` 정리

## 기본 개념

- 반복 조건을 먼저 검사
- 조건이 `true`인 동안 반복
- 조건이 처음부터 거짓이면 **한 번도 실행되지 않을 수 있음**

### 구조

```java
while (조건식) {
    내용
}
```

### 특징

- 반복 횟수가 명확하지 않을 때 적합
- 특정 조건에서 멈춰야 할 때 많이 사용
- 무한 반복은 `while (true)`

---

## 7. `do-while문` 정리

### 기본 개념

- **일단 한 번 실행 후**
- 조건을 검사

### 구조

```java
do {
    내용
} while (조건식);
```

### 특징

- 최소 1번은 실행이 보장됨
- 실무/기초 단계에서 상대적으로 사용 빈도는 낮음

### 예제

```java
int z = r.nextInt(10)+1;

do {
    System.out.println(z);
    z = r.nextInt(10)+1;
} while (z != 9);
```

## 8. `while문` / `do-while문` 비교 예제

```java
import java.util.Iterator;
import java.util.Random;

public class RMain3 {

	public static void main(String[] args) {

		// 1~ 10 사이의 랜덤한 정수를 출력
		// 9가 나오면 반복문을 종료!

		int x = 0;
		Random r = new Random();

		for (int i = 1; true; i++) {
			x = r.nextInt(10)+1;
			if (x == 9 ) {
				break;
			}
		}

		System.out.println(x);

		// 반복 횟수가 명확 - for 문(O)
		// 반복 조건 (if문의 반복 버전)
		// while문 : 조건을 먼저 검사.
		// 조건이 true(참)인 동안에 계속 반복해서 내용을 실행시킴.
		// 실행부분에서 조건을 false(거짓)으로 만들어서 멈추도록 하거나
		// break; 예약어로 반복문을 종료시킬 수 있다 !
		// 조건이 맞지 않으면 반복하던 작업을 중지 !
		// while : 반복 횟수를 정확히 알 수 없을 때,
		//         특별한 조건에서 멈춰야하는 경우에 사용!

		int y = r.nextInt(10)+1; // 랜덤한 정수 뽑기

		while (y != 9) { // 9가 아니면 ( = 9가 나올 때까지!)
			System.out.println(y);
			y = r.nextInt(10)+1; // 다시 while 조건으로 돌아가 체크
		}

		System.out.println("=====================");

		// do-while 문: 실행부터 하고 조건을 나중에 검사.
		// 한번은 반복이 보장이 됨.

		int z = r.nextInt(10)+1;

		do {
			System.out.println("//////////////////");
			System.out.println(z);
			z = r.nextInt(10)+1;
		} while (z != 9);
```

## 9. 합이 100을 넘는 순간 종료

### 문제

- `1 + 2 + 3 + ... + n`
- 합이 100을 넘는 순간 반복 종료
- 총합과 `n` 출력

### `while문` 버전

```java
		Random r1 = new Random();

		int sum = 0;
		int num = 0;

		while (sum <= 100 ) {
			num++;
			sum += num; // 값 누적

			System.out.println("//////////////");
			System.out.printf("n의 값: %d\n", num);
			System.out.printf("총 합: %d\n", sum);
			System.out.println("//////////////");
		}
```

### `for문` 버전

```java
		int sum2 = 0;
		int num2 = 0;

		for (int i = 0; true; i++) {
			num2++;
			sum2 += num2;

			if (sum2 > 100) {
				break;
			}
		}

		System.out.println("//////////////");
	}
}
```

### 정리 포인트

- 누적합이 특정 기준을 넘는 순간 종료할 때 `while`이 직관적
- `for`로도 가능하지만, 조건 제어는 `while`이 더 자연스러운 경우가 많음

# 11. `continue`, `break`, `return`

## 예제

```java
public class RMain4 {

	public static void main(String[] args) {

		// 반복문 10번 -> 반복문이 1번 돌때마다
		// 안녕하세요 x 3번 씩 출력
		// 갑자기 숫자 8이 맘에 안들어서

		String x = "안녕하세요";

		for (int i = 0; i < 10; i++) {
			if ( i == 8) {
				continue;
				// 8을 제외할 수 있다.
			}

			System.out.println(i);
			System.out.println(x);
			System.out.println(x);
			System.out.println(x);
		}

		System.out.println();
	}
}
```

---

## 개념 정리

### `continue;`

- **이번 턴만 종료**
- 아래 코드를 실행하지 않고 다음 반복으로 넘어감

### `break;`

- **반복문 자체 종료**
- 가장 가까운 반복문 또는 `switch` 종료

### `return;`

- **함수(메서드) 종료**
- 현재 메서드가 끝나므로 반복문도 함께 종료되는 효과

## 12. 중첩 반복문에서 `continue`, `break`

```java
import java.util.Scanner;

public class RMain4 {

	public static void main(String[] args) {

		String x = "안녕하세요";

		for (int i = 0; i < 10; i++) {
			if ( i == 8) {
				continue;
			}

			System.out.println(i);
			System.out.println(x);
			System.out.println(x);
			System.out.println(x);

			for (int j = 0; j < 10; j++) {
				if ( i % 2 == 0) {
					System.out.println("와 !");
				} else {
					//continue; // 강제반복 (이번턴 종료)
					// => 홀수일때는 그 턴 종료, 다음으로 넘어감.

					break; // (공식적), 반복문 끝 !
					//return; // (비공식적), 함수가 끝! 어쨌든 간에 반복문이 깨짐
				}
				System.out.println();
			}
		}

		System.out.println();
```

### 정리 포인트

- `break`는 **가장 가까운 반복문**만 종료
- 안쪽 `for (j...)`에서 `break`하면 바깥 `for (i...)`는 계속 진행

# 14. `switch`와 반복문 이름 붙이기(label)

## 예제

```java
import java.util.Scanner;

public class RMain5 {

	public static void main(String[] args) {

		// 누구:
		// 매니저
		// 시스템관리
		// 수업관련업무
		// 강사
		// 학생 관리
		// 수업관련 업무
		// 학생
		// 수업관련 업무
		// 를 반복하다가 끝이라고 입력하는 프로그램 종료

		Scanner s = new Scanner(System.in);
		String x = null;
		//String role = null;

		a : while(true) {
			// 반복문에 이름 붙이기 -> 반복문 앞에 [이름]:
			System.out.println("누구:");
			x = s.next();

			switch (x) {
			case "매니저":
				System.out.printf("시스템 관리\n수업관리\n");
				break;

			case "강사":
				System.out.printf("학생관리\n수업관련업무\n");
				break;

			case "학생":
				System.out.printf("수업관련업무\n");
				break;

			case "끝":
				break a;
				// break만 쓰면 가장 가까운 switch 종료
				// break 뒤에 반복문 이름을 붙이면 그 반복문까지 종료

			default:
				System.out.println("다시 입력!");
				break;
			}
		}
	}
}
```

### 정리 포인트

- `switch` 안에서 `break;` → `switch` 종료
- `break a;` → 이름이 `a`인 반복문 종료
- 반복문에 이름을 붙이면 중첩 구조에서 원하는 반복문을 정확히 종료 가능

# 15. 디버그와 label 예제

# 18. 핵심

## 1) `break`와 `continue`

- `break` : 반복문 종료
- `continue` : 이번 턴만 종료, 다음 턴 진행

## 2) `while`과 `do-while`

- `while` : 조건 먼저 확인
- `do-while` : 실행 먼저, 조건 나중 확인

## 3) 문자열 비교

```java
n.equals("ㅎㅇ")
```

- 문자열은 `==` 말고 `equals()` 사용

## 4) 무한 반복

```java
while (true) { }
for (;; ) { }
```

- 종료 조건을 내부에서 `break`로 처리

## 5) 중첩 반복문

- 일반 `break`는 가장 가까운 반복문만 종료
- 바깥 반복문까지 끝내려면 `label` 사용

# 암기용

## 1. `for문`

```java
for (초기값; 조건식; 증감식) {
    실행문
}
```

- 반복 횟수가 명확할 때 사용

---

## 2. `while문`

```java
while (조건식) {
    실행문
}
```

- 조건이 참인 동안 반복
- 조건을 먼저 검사
- 한 번도 실행 안 될 수 있음

---

## 3. `do-while문`

```java
do {
    실행문
} while (조건식);
```

- 무조건 1번은 실행
- 실행 후 조건 검사

---

## 4. `break`

```java
break;
```

- 반복문 즉시 종료

---

## 5. `continue`

```java
continue;
```

- 이번 턴만 종료
- 다음 반복으로 이동

---

## 6. 무한 반복

```java
while (true) { }
for (;; ) { }
```

- 내부에서 `break`로 종료

---

## 7. 문자열 비교

```java
str.equals("값")
```

- 문자열은 `==` 쓰지 말고 `equals()` 사용

---

## 8. 중첩 반복문 종료

```java
a: while (true) {
    break a;
}
```

- 라벨(label)을 붙이면 원하는 반복문 종료 가능

---

## 9. 반복문 선택 기준

- **횟수가 정해짐** → `for`
- **조건이 중요함** → `while`
- **무조건 1번 실행 필요** → `do-while`

- `while` = 조건 중심
- `do-while` = 일단 실행 후 검사
- `break` = 반복 종료
- `continue` = 이번 턴 스킵
- 문자열 비교 = `equals()`
- 바깥 반복문 종료 = `break 라벨명;`
