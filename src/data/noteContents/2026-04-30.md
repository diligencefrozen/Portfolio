### 값

Parameter
html/jsp
html에서 만들어낸 값(z,b)
request에 저장
String or String[]

attribute

java 에서 만들어낸 값
request에 저장
Object (객체)

### 이동

First ⇒ Second : 사용자가 뭔가 액션을 해서 넘어옴
		=request (요청)

Second => Third : 그냥 넘어감
 		- redirect : 단순 자동이동 (점검중 페이지 예제)
		response.sendRedirect("test.jsp");

forward : 자동이동 (값이 전달되는)
RequestDispatcher rd = request.getRequestDispatcher("Third.jsp");
rd.forward(request, response);

### RequestDispatcher : 클라이언트로부터 들어온 요청을 원하는 쪽으로 넘기는 기능

호출된 페이지에서는 request.setAttribute(key, value);
메소드를 통해서 넘겨받은 데이터를 처리할 수 있음.
⇒ Redirect와는 다르게 (데이터처리를 못하는 redirect) request, response 객체를 가지고 넘어가기 때문에

include : 포함 (Second 속에 Third가 포함되는)
위치조절 안됨 (Third가 무조건 상단에 들어옴) - 추천하진 않는 방식.

```java
package com.beaver.apr303.main;

import java.net.URLEncoder;
import java.net.http.HttpRequest;

import javax.servlet.http.HttpServletRequest;

import com.oreilly.servlet.MultipartRequest;
import com.oreilly.servlet.multipart.DefaultFileRenamePolicy;

public class Doctor {
	
	public static void calculate(HttpServletRequest req) {
		try {
			String path = req.getServletContext().getRealPath("img");
			System.out.println(path);
			MultipartRequest mr = new MultipartRequest(req, path, 30 * 1024 * 1024, "UTF-8", new DefaultFileRenamePolicy());
			
			String name = mr.getParameter("name");
			System.out.println(name);
			double height = Double.parseDouble(mr.getParameter("cm"));
			double weight = Double.parseDouble(mr.getParameter("kg"));
			height /= 100;
			double bmi = weight / (height * height);
			
			String result = "저체중";
			if (bmi >= 40) {
				result = "고도비만";
			} else if (bmi >= 35) {
				result = "중증도비만";
			} else if (bmi >= 30) {
				result = "경도비만";
			} else if (bmi >= 25) {
				result = "과체중";
			} else if (bmi >= 18.5) {
				result = "정상";
			}
			
			// BMI 수치 => 소수점 둘째자리까지만
			String bmi2 = String.format("%.2f",  bmi);
			double bmi3 = Double.parseDouble(bmi2);
			
			String image = mr.getFilesystemName("photo");
			image = URLEncoder.encode(image, "UTF-8");
			image = image.replace("+", " ");
			
			// 객체를 만들어서 한꺼번에 보내자!
			Patient p = new Patient(name, height, weight, bmi3, image, result);
			//멀티파트 리퀘스트 클래스에는 setattribute 메소드가 없음
			
			// Httpservlet class 를 활용해서만
			//attribute 를 생성해낼수 있음.
			req.setAttribute("p", p);
		} catch (Exception e) {
			// TODO: handle exception
			e.printStackTrace();
		}
	}

}

```
