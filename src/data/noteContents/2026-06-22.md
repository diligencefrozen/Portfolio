## 1. 정리

Pandas는 표 형태 데이터를 다루는 라이브러리이며, 데이터 분석의 기본 도구다.

---

# 2. Series

## Series란?

Series는 **1차원 데이터 구조**다.

하나의 열(column)처럼 생각하면 된다.

```python
import pandas as pd

s = pd.Series([4, 5, -4, -5])
```

인덱스를 직접 지정할 수도 있다.

```python
s = pd.Series([4, 5, -4, -5], index=['a', 'b', 'c', 'd'])
```

## 값 선택

```python
s['a']
s[['a', 'c']]
s['a':'c']
```

## 조건 필터링

```python
s[s > 0]
```

## 결측값 확인

```python
s.isnull()
s.notnull()
```

---

# 3. DataFrame

## DataFrame이란?

DataFrame은 **2차원 표 형태 데이터 구조**다.

여러 개의 Series가 모인 형태다.

```python
df = pd.DataFrame({
    'name': ['A', 'B'],
    'score': [90, 80]
})
```

## 행/열 이름 변경

```python
df.rename(columns={'score': 'math'}, inplace=True)
```

`inplace=True`는 원본에 바로 적용한다.

---

# 4. 행과 열 선택

## loc / iloc

| 함수 | 기준 |
| --- | --- |
| `loc` | 이름 기준 |
| `iloc` | 위치 번호 기준 |

```python
df.loc[0]
df.iloc[0]
```

특정 값 선택:

```python
df.loc[0, 'name']
df.iloc[0, 1]
```

## 열 선택

```python
df['name']
df[['name', 'score']]
```

---

# 5. 행/열 삭제

```python
df.drop('행이름', axis=0)
df.drop('열이름', axis=1)
```

| axis | 의미 |
| --- | --- |
| `axis=0` | 행 |
| `axis=1` | 열 |

열 삭제할 때는 꼭 `axis=1`을 쓴다.

---

# 6. 행/열 추가

## 열 추가

```python
df['new_col'] = [1, 2, 3]
```

## 행 추가

```python
df.loc[3] = ['Kim', 100]
```

새로운 인덱스를 쓰면 행이 추가된다.

---

# 7. 슬라이싱

```python
df.iloc[0:2]
df.iloc[::2]
df.iloc[::-1]
```

## 주의

틀린 코드:

```python
df.iloc[[0:2]]
```

올바른 코드:

```python
df.iloc[0:2]
```

또는:

```python
df.iloc[[0, 1]]
```

# 8. Series / DataFrame 연산

## Series와 숫자

```python
s / 200
```

모든 값에 연산이 적용된다.

## Series끼리 연산

```python
s1 + s2
```

같은 인덱스끼리 계산된다.

## 결측값 대체 후 연산

```python
s1.add(s2, fill_value=0)
```

`fill_value=0`은 없는 값을 0으로 채운 뒤 계산한다.

# 12. 평균과 중앙값

## 평균

전체 값을 더한 뒤 개수로 나눈 값.

```
평균 = 전체 합 / 개수
```

## 중앙값

데이터를 크기순으로 정렬했을 때 가운데 값.

극단값이 있으면 평균보다 중앙값이 더 현실적일 수 있다.

---

# 13. 상관계수

```python
df.corr(numeric_only=True)
```

상관계수는 두 변수의 관계를 나타낸다.

| 값 | 의미 |
| --- | --- |
| 1에 가까움 | 양의 상관 |
| -1에 가까움 | 음의 상관 |
| 0에 가까움 | 관계 약함 |

상관관계는 인과관계를 의미하지 않는다.
