### 핵심 개념

결측치 = 비어 있는 데이터

Pandas에서는 주로 `NaN`으로 표시됨.

### 사용 데이터

```python
t = sns.load_dataset('titanic')
```

### 전체 데이터 정보 확인

```python
t.info()
```

확인 가능한 것:

- 행 개수
- 열 개수
- 각 컬럼의 결측치 여부

## 범주형 데이터

| 구분 | 설명 | 예시 |
| --- | --- | --- |
| 순위형 | 순서에 의미 있음 | A+, A, B, C |
| 명목형 | 순서에 의미 없음 | 성별, 혈액형, 도시 |

## 평균 컬럼 만들기


### 핵심

| 코드 | 의미 |

| `.mean(axis=1)` | 행 기준 평균 |
| `round(..., 2)` | 소수점 둘째 자리까지 |

---

## 구간 나누기

### 1단계: 구간 경계 만들기

```python
count, bins_dividers = np.histogram(df['avg'], bins=3)
```

결과:

```python

bins_dividers
```

- `count`: 각 구간에 들어간 데이터 개수
- `bins_dividers`: 구간 경계값

---

### 2단계: 점수 구간 라벨링

```python
bins_names = ['low', 'middle', 'high']

df['a_bin'] = pd.cut(
    x=df['avg'],
    bins=bins_dividers,
    labels=bins_names,
    include_lowest=True
)
```

### 결과

평균 점수에 따라 `low`, `middle`, `high`로 분류됨.

### 핵심 패턴

```python

pd.cut()
```

## Period

`Period` = 특정 기간

```python
p = pd.Period('2026-05-23')
```


### Timestamp vs Period

| 구분 | 의미 |

| `Timestamp` | 하나의 시점 |
| `Period` | 시작~끝이 있는 기간 |

## 4. 함수 매핑

> 파일: `Jun23_4_FunctionMapping.ipynb`
> 

## 함수 매핑이란?

Series나 DataFrame의 각 요소에 함수를 1:1로 적용하는 것.

### 장점

- 반복문보다 간단함
- 사용자 정의 함수 적용 가능
- lambda 함수도 사용 가능
- 데이터 전처리에 자주 사용됨

---

## 합계 컬럼 만들기

```python
ms['sum'] = ms[['kor', 'eng', 'math']].sum(axis=1)
```

### 핵심

```python
axis=1
```

행 기준으로 계산한다는 뜻.

---

## 사용자 정의 함수 만들기

```python
def divide_three(a):
    return a / 3
```

---

## apply로 함수 적용

```python
ms['avg'] = round(ms['sum'].apply(divide_three), 2)
```

### 흐름

```python
sum 컬럼 생성
→ divide_three 함수 생성
→ apply로 각 값에 함수 적용
→ avg 컬럼 생성
```

---

## 더 간단한 방식

```python
ms['avg'] = round(ms[['kor', 'eng', 'math']].mean(axis=1), 2)
```

### 비교

| 방식 | 특징 |
| --- | --- |
| `sum + apply()` | 함수 매핑 연습에 좋음 |
| `mean(axis=1)` | 실제로는 더 간단하고 직관적 |

---

## 주의할 점

```python
def divide_three(a):
    return a / 3

    print(divide_three(9))
```

`return` 아래에 있는 코드는 실행되지 않음.

올바른 테스트:

```python
def divide_three(a):
    return a / 3

print(divide_three(9))
```

```python
li.select_one('.listPhoto a')['href']
```

## 리스트에 데이터 저장

```python
menuList = []
urlList = []

for li in ulTag.select('li'):
    menuList.append({

        '업종': li.select_one('.listType').text,
        '지역': li.select_one('.listArea').text

    urlList.append(li.select_one('.listPhoto a')['href'])
```

### 구조

```python
menuList = [{가게 정보}, {가게 정보}, ...]
urlList = [상세페이지 URL, 상세페이지 URL, ...]
```
