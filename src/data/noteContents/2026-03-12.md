# 1. Switch - Case (조건문)

### 개념

`switch` 문은 **특정 값에 따라 여러 경우를 나누어 실행**할 때 사용한다.

보통 `if - else` 보다 **가독성이 좋은 경우**가 많다.

예시 문제

점수를 입력받아 등급 출력

| 점수 | 등급 |
| --- | --- |
| 90 이상 | A |
| 80 이상 | B |
| 70 이상 | C |
| 60 이상 | D |
| 그 외 | F |

---

## 비효율적인 방법

점수 하나하나를 case로 나누는 방식

```java
switch (answer) {

case 100: case 99: case 98: case 97: case 96:
case 95: case 94: case 93: case 92: case 91: case 90:
System.out.println("A");
break;

case 89: case 88: case 87: case 86: case 85:
case 84: case 83: case 82: case 81: case 80:
System.out.println("B");
break;

case 79: case 78: case 77: case 76: case 75:
case 74: case 73: case 72: case 71: case 70:
System.out.println("C");
break;

case 69: case 68: case 67: case 66: case 65:
case 64: case 63: case 62: case 61: case 60:
System.out.println("D");
break;

default:
System.out.println("F");
break;

}
```

이 방법은 **코드 길이가 지나치게 길어지고 유지보수가 어렵다.**

---

## 효율적인 방법

점수를 **10으로 나눈 값**을 기준으로 분류

```java
import java.util.Scanner;

public class YMain1 {

public static void main(String[] args) {

Scanner s = new Scanner(System.in);

System.out.println("입력:");

int answer = s.nextInt();

System.out.printf("사용자에게서 받아온 데이터:%d점\n", answer);

String grade = "";

switch (answer / 10) {

case 10:
case 9:
grade = "A";
break;

case 8:
grade = "B";
break;

case 7:
grade = "C";
break;

case 6:
grade = "D";
break;

default:
grade = "F";
break;

}

System.out.printf("당신은 %s 등급입니다.\n", grade);

}

}
```

### 핵심

```
95 / 10 = 9
83 / 10 = 8
74 / 10 = 7
```

→ **10단위 그룹으로 분류**

이런 방식은

**"데이터 정규화 또는 범주화 전략"**

이라고 볼 수 있다.

# 2. 함수 (Method)

함수는 **특정 작업을 수행하는 코드 묶음**이다.

### 장점

- 코드 재사용 가능
- 가독성 증가
- 유지보수 쉬움

---

## 함수 구조

```
접근제어자 반환타입 함수이름(매개변수) {

코드

return 값;

}
```

---

# 예제 코드

```java
import java.util.Scanner;

public class YMain2 {

public static int num() {

Scanner s = new Scanner(System.in);

System.out.println("정수 입력:");

int answer = s.nextInt();

System.out.printf("사용자에게서 받아온 데이터: %d\n", answer);

return answer;

}
```

### 설명

```
public static int num()
```

의미

| 요소 | 의미 |
| --- | --- |
| public | 어디서든 접근 가능 |
| static | 객체 생성 없이 사용 |
| int | 반환값 타입 |
| num | 함수 이름 |

---

## 양수 / 음수 / 0 판별 함수

```java
public static void calc(int answer) {

if (answer > 0) {
System.out.println("양");

} else if (answer == 0) {
System.out.println("0");

} else {
System.out.println("음");
}

}
```

# 3. 배수 판별 함수

문제

정수를 입력받아

| 조건 | 출력 |
| --- | --- |
| 3과4 공배수 | 3의 배수 + 4의 배수 |
| 3의 배수 | 3의 배수 |
| 4의 배수 | 4의 배수 |
| 둘다 아님 | 몰라 |

---

```java
public static String judge() {

Scanner s = new Scanner(System.in);

System.out.println("정수 입력:");

int answer2 = s.nextInt();

if (answer2 == 0) {
return "0은 모든 정수의 배수";

}
else if (answer2 % 12 == 0) {
return "3의 배수 + 4의 배수";

}
else if (answer2 % 3 == 0) {
return "3의 배수";

}
else if (answer2 % 4 == 0) {
return "4의 배수";

}

return "몰라 !!!";

}
```

### 중요한 개념

```
3과 4의 공배수 = 12의 배수
```

즉

```
x % 12 == 0
```

이 가장 효율적인 방법이다.

# 4. 반복문 (Loop)

### 반복문의 목적

어떤 작업을 **여러 번 반복 실행**할 때 사용한다.

대표적인 경우

1. 횟수 반복
2. 조건 반복

---

# For 문

## 기본 구조

```java
for (초기화; 조건식; 증감식) {

반복 실행 코드

}
```

### 실행 과정

1. 변수 초기화
2. 조건 검사
3. 코드 실행
4.  증감
5. 다시 조건 검사
6. 조건이 거짓이면 종료

---

# 예제 1 (카운트다운)

```java
for (int i = 5; i > 0; i--) {

System.out.println(i);

}
```

출력

```
5
4
3
2
1
```

---

# 예제 2 (홀수 출력)

1 ~ 21 홀수 출력

### 방법 1

```java
for (int i = 1; i < 22; i++) {

if (i % 2 == 1) {
System.out.println(i);
}

}
```

---

### 방법 2 (더 효율적)

```java
for (int j = 1; j < 22; j += 2) {

System.out.println(j);

}
```

이 방법이 **연산량이 적다.**

# 5. Random 숫자 생성

```java
import java.util.Random;

Random r = new Random();
```

1 ~ 10 랜덤

```java
int x = r.nextInt(10) + 1;
```

---

### 반복문 안에서 변수 생성 (X)

```java
for (int i = 0; i < 5; i++) {

int x = r.nextInt(10) + 1;

}
```

→ 반복마다 **새 변수 생성**

---

### 반복문 밖에서 생성

```java
int x = 0;

for (int i = 0; i < 5; i++) {

x = r.nextInt(10) + 1;

}
```

→ 메모리 효율이 더 좋다.

# 6. 팩토리얼

```
7! = 7 × 6 × 5 × 4 × 3 × 2 × 1
```

---

```java
int mul = 1;

for (int i = 1; i <= 7; i++) {

mul *= i;

System.out.println(mul);

}
```

### 핵심

초기값이 **1이어야 한다**

```
0이면 모든 곱이 0이 된다
```

# 7. 조건 합 계산

문제

1 ~ 50 중에서

**3 또는 5의 배수 합**

---

```java
int sum = 0;

for (int i = 1; i <= 50; i++) {

if (i % 3 == 0 || i % 5 == 0) {

sum += i;

}

}

System.out.println(sum);
```

결과

```
593
```

# 8. 구구단 (중첩 반복문)

```java
for (int x = 1; x <= 9; x++) {

for (int dan = 2; dan <= 9; dan++) {

System.out.printf("%d x %d = %d\t", dan, x, dan * x);

}

System.out.println();

}
```

```
for {
    for {
    }

```

# 9. 별 찍기 (패턴 문제)

### 예제 1

```
ㅋ
ㅋㅋ
ㅋㅋㅋ
ㅋㅋㅋㅋ
ㅋㅋㅋㅋㅋ
```

```java
for (int i = 0; i < 5; i++) {

for (int j = 0; j <= i; j++) {

System.out.print("ㅋ");

}

System.out.println();

}
```

---

### 예제 2

```
ㅋㅋㅋㅋㅋ
ㅎㅎㅎㅎ
ㅋㅋㅋ
ㅎㅎ
ㅋ
```

```java
for (int i = 0; i < 5; i++) {

for (int j = 0; j < (5 - i); j++) {

System.out.print((i % 2 == 0) ? "ㅋ" : "ㅎ");

}

System.out.println();

}
```
