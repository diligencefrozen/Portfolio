## 1-1. `Test4.java`

### 코드

```java
package com.beaver.today.test;

import java.sql.Connection;
import java.sql.PreparedStatement;
import java.sql.ResultSet;
import java.util.Scanner;

import com.jeesung.db.manager.JeesungDBManager;

public class Test4 {
	public static void main(String[] args) {
		Connection con = null;
		ResultSet rs = null;
		PreparedStatement pstmt = null;

		try {
			con = JeesungDBManager.connect();

			// 1. 카테고리 목록 출력
			String sql = "select c_id, c_name from category order by c_id";
			pstmt = con.prepareStatement(sql);
			rs = pstmt.executeQuery();
			System.out.println("====== 카테고리 목록 ======");
			while (rs.next()) {
				System.out.printf("%d) %s\n",
						rs.getInt("c_id"),
						rs.getString("c_name"));
			}
			System.out.println("======================");

			// 2. 카테고리 이름 입력
			// => 그에 맞는 c_id 값을 가져오기
			// => p_c_id에 연결
			Scanner k = new Scanner(System.in);
			System.out.print("카테고리 이름(부분 작성 X) :");
			String cName = k.next();
			sql = "select c_id from category where c_name = ?";
			pstmt = con.prepareStatement(sql);
			pstmt.setString(1,  cName);
			rs = pstmt.executeQuery();

			int categoryID = 0;
			while (rs.next()) {
				categoryID = rs.getInt("c_id");

			}
			// 3. 상품명 입력
			System.out.println("상품명 입력:");
			String productName = k.next();

			// 4. 가격 입력
			System.out.println("가격 입력:");
			int price = k.nextInt();

			// 5. Insert
			sql = "insert into product "
					+ "(p_id, p_c_id, p_name, p_price) "
					+ "values(product_seq.nextval, "
					+ "?, ?, ?)";
			pstmt = con.prepareStatement(sql);
			pstmt.setInt(1, categoryID);
			pstmt.setString(2, productName);
			pstmt.setInt(3,  price);

			if (pstmt.executeUpdate() == 1) {
				System.out.println("Product 등록 성공 !");
			}
		} catch (Exception e) {
			// TODO: handle exception
			e.printStackTrace();
		} finally {
			JeesungDBManager.close(con, pstmt, rs);
		}
	}

}
```

## 1-2. `Test5`

### 코드

```java
package com.beaver.today.test;

import java.sql.Connection;
import java.sql.PreparedStatement;
import java.sql.ResultSet;
import java.text.SimpleDateFormat;
import java.util.Scanner;

import com.jeesung.db.manager.JeesungDBManager;

public class Test5 {
	public static void main(String[] args) {

		Connection con = null;
		PreparedStatement pstmt = null;
		ResultSet rs = null;

		try {
			con = JeesungDBManager.connect();

			Scanner k = new Scanner(System.in);
			System.out.println("카테고리 검색: ");
			String categoryName = k.next();

			String sql = "select p.p_id, c.c_name, p.p_name, "
					+ "p.p_price, p.p_created_at "
					+ "from product p, category c "
					+ "where c.c_id = p.p_c_id "
					+ "and c.c_name like '%'||?||'%' "
					+ "order by p.p_created_at desc, "
					+ "p.p_id desc";

			pstmt = con.prepareStatement(sql);
			pstmt.setString(1,  categoryName);

			rs = pstmt.executeQuery();
			System.out.println("P_ID | C_NAME | P_NAME | P_PRICE | P_CREATE_AT");
			System.out.println("================================================");

			SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
			while (rs.next()) {
				int id = rs.getInt("p_id");
				String cName = rs.getString("c_name");
				String pName = rs.getString("p_name");
				int price = rs.getInt("p_price");
				String createdAt = sdf.format(rs.getDate("p_created_at"));
				System.out.printf("%,d | %s | %s |%,d | %s\n", id, cName, pName, price, createdAt);
				System.out.println("================================================");
			}
		} catch (Exception e) {
			// TODO: handle exception
			e.printStackTrace();
		} finally {
			JeesungDBManager.close(con, pstmt, rs);
		}
	}

}
```

## 4-6. `<input>` 정리

### type

```
hidden : 사용자한테 안보임, 서버로 값들을 보낼 때 사용
text : 한 줄 짜리 텍스트 입력
search : 검색
password : 비밀번호 입력
range : 숫자를 조절할 수 있는 슬라이드 막대
number : 숫자를 조절 할 수 있는 화살표
checkbox : 체크박스 (여러개  선택 가능 O)
radio : 라디오 버튼 (1개만 선택 가능 O)
image, submit, reset, button, file, ...
```

### 속성

```
readonly : 읽기 전용으로 만들기
placeholder : 힌트 표시 (input 클릭하면 내용 사라짐)
autofocus : 그 위치에 커서가 표시됨

autocomplete : 자동완성
maxlength : 최대로 입력할 수 있는 글자 수 설정
value : 기본값 설정
```

- `name` 속성이 실제 폼 전송에서 매우 중요함
- `id`는 JS/CSS 선택 및 label 연결에 자주 사용
- `placeholder`는 안내 문구일 뿐 실제 값이 아님

## 4-11. `h6_table.html`

### 코드

```html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Insert title here</title>
</head>
<body>

	<!--
		<table> : HTML 에서 표를 만들 때 사용하는 태그'
			기본 : <tr>, <th>, <td>
			tr은 'table row'의 약자, 테이블의 행을 표현
			td는 'table data'의 약자,
				테이블에 들어가는 데이터, 셀을 의미
			즉, 하나의 테이블 안에 <tr> 태그로 행을 표현하고,
			그 안에 <td> 태그를 이용하여 데이터를 표현하는 형태
			th는 'table header' 의 약자, 컬럼의 타이틀을 표현
			속성 : border, width, height (table)
					align, valign (정렬 방법 td, th)
					colspan, rowspan (칸 합병, td)

	 -->

	 <table border = "1" width = "300" height = "300">
	 	<tr>
	 		<td align ="center">1</td>
	 		<td>2</td>
	 	</tr>
	 	<tr>
	 		<td>3</td>
	 		<td>4</td>
	 	</tr>
	 </table>

	 <hr>

	 <table border = "1" width = "300" height ="300">
	 <tr>
	 	<td align ="center">ㄱ</td>
	 	<td valign="baseline" align="center">ㄹ</td>
	 </tr>

	 <tr>
	 	<td valign="baseline" align="center">ㄴ</td>
	 	<td valign="bottom" align="center">ㅁ</td>
	 </tr>

	 <tr>
	 	<td valign="top" align="right">ㄷ</td>
	 	<td valign="bottom" align="left">ㅂ</td>
	 </tr>
	 </table>
	 <hr>

	 <!--
	 	셀을 가로로 합병하려면 1열, 2열을 합병해야하니 colspan
	 	셀을 세로로 합병하려면 1행, 2행을 합병해야하니, rowspan
	  -->
	 <table border ="1" width = "300" height ="300">
	 	<tr>
	 		<td colspan ="2">1</td>

	 		<td rowspan ="2">3</td>
	 	</tr>

	 	<tr>
	 		<td rowspan = "2">4</td>
	 		<td>5</td>

	 	</tr>

	 	<tr>

	 		<td colspan ="2">8</td>

	 	</tr>

	 </table>

</body>
</html>
```

- 병합(`colspan`, `rowspan`)은 표 설계 시 자주 나오는 핵심 포인트!
