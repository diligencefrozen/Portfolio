### Exception 처리할 수 있는 방법

→ `try-catch`

```java

       일단 이 부분을 실행 - (내용)

} catch (예외유형명 변수명) {

      try쪽을 실행하다가 해당 유형의 예외가 발생하면 이 부분을 실행

      만약, 예외 없이 정상적으로 실행됐을 경우,

       catch 부분은 그냥 지나치고 그 다음 코드를 수행

      …

} finally {

    정상적으로 프로그램이 실행됐던지,

     예외가 터졌던지 상관없이 실행.

     return보다 먼저 실행

}
```

# Wrapper Class

## 개념

Wrapper Class : (자료형) 기본형들의 객체형 (포장객체)

→ `Integer`, `Double`, `Long`, `Void`, `Boolean`, …

기본 자료형을 객체로 다루기 위해 사용하는 Class.

기본형이 가지고 있는 값을 내부에 두고, 그 위를 포장하는 방식

### 용어

- Boxing : 기본형 ⇒ 객체형
- UnBoxing : 객체형 ⇒ 기본형

### 사용하는 이유

1. 객체나 클래스가 제공하는 method 사용할 때
2. 클래스가 제공하는 상수를 사용 (`MIN_VALUE`, `MAX_VALUE`, …)
3. 숫자, 문자로 형 변환을 할 때

---

```java
String b = (String) a;
```

⇒ `(기본형 => 참조형)` 이게 안됨.


## 자주 쓰는 메서드

### `charAt()`

특정 위치에 있는 글자를 뽑아낼 때

```java
System.out.println(s2.charAt(2));
```

### `length()`

총 글자 수를 알고 싶을 때

```java
System.out.println(s2.length()+"개");
```

### `startsWith()`

시작하는 단어가 맞는지 확인할 때 (`true / false`)

```java
System.out.println(s2.startsWith("곧"));
```

### `contains()`

찾는 글자가 있는지 (`true / false`)

```java
System.out.println(s2.contains("너무"));
```

### `replace()`

특정 글자를 다른 글자로 바꿀 때

```java
System.out.println(s2.replace("4월", "5월"));
```

### `substring()`

원하는 위치의 글자를 뽑을 때

(시작위치, 끝위치 - 1)

```java
System.out.println(s2.substring(2, 6 -1));
```

### `subSequence()`

원하는 위치의 글자를 뽑을 때

(시작위치, 끝위치 -1)

```java
System.out.println(s2.subSequence(2, 6 -1));
```

### `String.format()`

String 객체를 생성하는데 형식을 잡아서

```java
String s3 = String.format("무게 : %.2fkg",123.456789);

System.out.printf(s3);
```

# 암기용

## 1) error / warning / exception 차이

- **error** = 문법 틀림 → 컴파일 불가
- **warning** = 실행은 되지만 코드 상태가 안 좋음
- **exception** = 실행 중 문제 발생

## 2) try-catch-finally

- **try** = 일단 실행
- **catch** = 예외 나면 잡기
- **finally** = 예외 나든 말든 무조건 실행

## 3) 자주 나오는 예외

- `NullPointerException` = null인데 `.찍음`
- `ArrayIndexOutOfBoundsException` = 배열 범위 초과
- `NumberFormatException` = 문자열을 숫자로 못 바꿈
- `ArithmeticException` = 0으로 나누기

## 4) Wrapper Class

- 기본형을 객체처럼 쓰기 위한 클래스
- 대표: `Integer`, `Double`, `Boolean`
- 자주 쓰는 것
    - `Integer.parseInt()`
    - `Double.parseDouble()`
    - `Integer.MAX_VALUE`
    - `Integer.MIN_VALUE`

## 5) 날짜

- 옛날 방식: `Date`, `Calendar`
- 최신 방식: `LocalDate`, `LocalTime`, `LocalDateTime`
- `Date`는 오래돼서 `deprecated` 관련 내용 자주 나옴

## 6) 문자열

- 한 글자 꺼내기 → `charAt()`
- 길이 → `length()`
- 포함 여부 → `contains()`
- 바꾸기 → `replace()`
- 자르기 → `substring()`
- 분리하기 → `split()`, `StringTokenizer`
